# Supporting Materials

Upload-ready files for the Sentient Foundation Open Source AGI Grant application.

Recommended upload:

1. `Open_Agent_Safety_Kit_Grant_Deck_Final.pdf` — polished grant deck
2. `Open_Agent_Safety_Kit_Grant_Deck_Final.pptx` — editable version
3. `Open_Agent_Safety_Kit_One_Pager.pdf` — optional short summary

The PDF deck is for Typeform upload. The PPTX is for editing in PowerPoint or Google Slides if changes are needed.
